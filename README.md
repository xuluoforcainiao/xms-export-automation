# xms-export-automation

Automates the XMS customer service abnormal item export workflow. Covers SSO login, clearing the shelf organization filter, exporting abnormal items, polling the download center, downloading the result file, and optionally uploading to DingTalk. Use when the user needs to export XMS abnormal items, set up scheduled XMS exports, or manage XMS data delivery.
